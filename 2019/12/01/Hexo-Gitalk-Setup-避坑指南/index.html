Hexo Gitalk Setup && 避坑指南

Posted on 2019-12-01 In Hexo

This tutorial is based on you already setupped Hexo website.

我的配置:
Hexo: v4.0.0
NexT: v7.5.0

网上很多Gitalk配置文档都写了如何配置新的swig，但新版本的Next主题自带Gitalk配置文件，不用额外配swig脚本。

一、用 GitHub 账号创建 Application

用以生成 Client ID 和 Client Secrete，使用户能够有权限操作Git仓库，生成Issue(留言)
闭坑：注意，此处创建的是 OAuth Apps，而不是GitHub Apps！如果创建成GitHub Apps后期会无法评论。

在你的GitHub主页中，从右上角头像进入Setting界面，在左侧边栏中点击Developer Settings，选择左侧第二个选项OAuth Apps，点击New OAuth App。
